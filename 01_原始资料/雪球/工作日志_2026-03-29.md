## 🛠️ 技术方案

### 挑战
1. **JavaScript 动态加载** — 雪球使用 React，内容动态渲染
2. **反爬虫机制** — 需要真实浏览器环境
3. **登录要求** — 某些内容需要登录

### 解决方案

#### 方案 A：Camoufox（推荐）
- 反爬虫浏览器，可绕过大多数反爬机制
- 需要安装：`brew install camoufox`
- 优点：稳定、可靠
- 缺点：需要额外安装

#### 方案 B：OpenClaw 浏览器自动化
- 使用 Gateway API 调用浏览器工具
- 脚本：`xueqiu_monitor_v2.py`
- 优点：无需额外安装
- 缺点：可能被反爬虫检测

#### 方案 C：手动导出
- 手动访问页面，复制内容
- 保存为 JSON 文件
- 优点：100% 可靠
- 缺点：无法自动化
