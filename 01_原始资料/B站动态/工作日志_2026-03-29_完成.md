## ⚠️ 遇到的问题

### 1. HTTP 412 限流（关键阻塞）

**现象：**
- 首次抓取（12:26-12:28）：触发限流，仅收录 1 条
- 第二次尝试（12:31-12:33）：全量 412，无法继续
- 第三次尝试（14:46-14:50）：指数退避重试 5 次，仍无法突破

**原因分析：**
- B站反爬机制：连续高频请求触发 IP 级封禁
- 评论 API 限流独立于动态 API
- 需要等待 30 分钟~几小时自然解除

**解决方案：**
- ✅ 设置定时重试任务（15:40、16:10、16:12 等）
- ✅ 指数退避策略（5s→10s→20s→40s→80s）
- ✅ 20:27 限流自然解除，成功抓取

### 2. 脚本挂死问题

**现象：**
- 多次运行脚本无输出，进程卡住
- 日志文件更新缓慢

**原因：**
- 指数退避等待时间过长（最后一次等待 80 秒）
- 脚本在后台运行，无实时反馈

**改进方案：**
- 改用 `--force-full` 参数强制全量抓取
- 增加日志刷新频率
- 考虑改用浏览器自动化绕过 API 限流
