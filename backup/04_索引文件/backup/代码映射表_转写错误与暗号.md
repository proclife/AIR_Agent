## 三、⚠️ 需要特殊处理的项

### 3.1 子串级联问题

**藏矿 → 西藏矿业**（52处残留）

- **问题**：藏矿 是 "西藏矿业" 的子串，直接替换会导致 "西西藏矿业业"
- **处理方案**：使用 `(?<!西)` 负向前瞻匹配，仅替换"独立藏矿"
- **风险**：部分"藏矿"可能确实是"西藏矿业"的口语简称（非子串），需要逐条验证

**液冷 → AI液冷**（33处残留）

- **问题**：液冷 是 "AI液冷" 的子串，且存在 "液冷液冷" 级联产物
- **处理方案**：
  1. 先修级联：`液冷液冷` → `液冷`，`冷液冷` → `液冷`
  2. 再替换：`液冷` → `AI液冷`（使用 `(?<!AI)` 负向前瞻）
- **验证**：当前可读版 "AI液冷" 有33处，"液冷" 也有33处 → 说明33处独立"液冷"未被替换

**冷子 → 曙光数创**（2处残留）

- **问题**：冷子 可能出现在 "冷子液冷" 等词组中，替换后得到 "曙光数创液冷"（语义正确）
- **处理方案**：直接全词替换 `冷子` → `曙光数创`
