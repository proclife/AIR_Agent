## 二、核心推理模式

### 2.1 决策树模型

```
第一步：看走势
├── 突破 → 进入第二步
└── 未突破 → 等待

第二步：验逻辑
├── 行业景气度？
├── 基本面支撑？
└── 政策支持？

第三步：定仓位
├── 高位股 → 轻仓试探
├── 低位股 → 重仓持有
└── 震荡市 → 分批建仓

第四步：设止损
├── 高位破线 → 立即止损
├── 低位熬着 → 不慌
└── 逻辑验证 → 可加仓
```

### 2.2 买卖决策矩阵

| 情况 | 走势 | 基本面 | 操作 |
|------|------|--------|------|
| A | 突破 | 向好 | 买入 |
| B | 突破 | 向差 | 观望 |
| C | 破位 | 向好 | 等待 |
| D | 破位 | 向差 | 止损 |

### 3.1 预测输入特征

| 特征 | 描述 | 莫大权重 |
|------|------|----------|
| 趋势强弱 | 五日线/十日线/五十日线位置 | 极高 |
| 行业景气 | 政策+资金+业绩 | 高 |
| 估值水平 | PE/PB历史分位 | 中 |
| 仓位变化 | 陆股通/机构持仓 | 中 |
| 宏观环境 | 流动性/风险偏好 | 中 |

### 3.2 持仓调整规律

**加仓信号**：
- 突破重要均线（五日线）
- 行业政策利好出台
- 业绩超预期
- 大盘回暖

**减仓信号**：
- 高位破线
- 逻辑证伪
- 出现更优替代
- 大盘系统性风险

## 四、思维链模板（可直接使用）

### 4.1 选股模板
```
1. 发现异动
   └─ 什么股票在涨？涨了多少？
   
2. 趋势验证
   └─ 是否突破均线？突破量能如何？
   
3. 逻辑挖掘
   └─ 为什么涨？政策/业绩/概念？
   
4. 风险评估
   └─ 下跌空间？止损位？
   
5. 仓位决策
   └─ 多少仓位？分批还是一次？
```

### 4.2 持仓检查模板
```
每日检查：
├── 核心股走势？
├── 趋势是否破坏？
├── 有无新信息？
└── 需要调仓吗？

每周复盘：
├── 本周操作对错？
├── 逻辑是否变化？
└── 下周计划？
```
