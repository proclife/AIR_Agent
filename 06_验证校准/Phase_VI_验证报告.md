## 三、框架改进建议

### 3.1 增加高位警戒

当股票处于历史高位（>80%分位）时：
- 即使五日线上，也要降低置信度
- 建议改为"谨慎持有"或"考虑减仓"

### 3.2 增加低位买入条件

当股票同时满足：
- 历史低位（<20%分位）
- 基本面无致命问题
- 困境反转逻辑存在

→ 建议提升为"重点关注"

### 3.3 综合评分调整

| 条件 | 分数调整 |
|------|----------|
| 历史高位（>80%）| -1 |
| 历史低位（<20%）| +1 |
| 行业景气度高 | +1 |
| 行业景气度低 | -1 |

## 四、置信度校准

### 4.1 当前置信度体系

| 评分 | 含义 | 实际操作 |
|------|------|----------|
| 9-10/10 | 高度确定 | 可重仓 |
| 7-8/10 | 较强确定 | 可加仓 |
| 5-6/10 | 中性 | 轻仓观望 |
| 3-4/10 | 较弱 | 观察 |
| 1-2/10 | 低确定性 | 不操作 |

### 4.2 校准建议

**需要降低置信度的情况**：
- 历史高位（>80%分位）- 建议降2级
- 无行业景气度支撑 - 建议降1级
- 量能萎缩 - 建议降1级

**可以提高置信度的情况**：
- 历史低位（<20%分位）- 建议升1级
- 行业景气度高 - 建议升1级
- 放量突破 - 建议升1级
