# HEARTBEAT.md — 莫大韭菜市场监控系统

---

## 🎯 监控目标

基于莫大韭菜的投资框架，监控以下内容：

1. **持仓股走势** — 西藏矿业、航天动力、比亚迪、智光电气、华大智造、曙光数创、华钰矿业、司尔特、海目星
2. **关注板块** — 新能源车、AI人工智能、锂矿、商业航天、储能、半导体
3. **大盘环境** — 上证指数、创业板、沪深300
4. **宏观信号** — 美联储政策、流动性、汇率
5. **重大新闻** — 政策利好/利空、行业突破性进展

## 🚨 预警规则

### 红色预警（立即通知）

| 触发条件 | 应对 |
|---------|------|
| 持仓股跌幅 > 5% | 检查是否破线，考虑止损 |
| 持仓股跌破五日线 | 准备减仓/止损 |
| 大盘暴跌 > 3% | 评估系统性风险 |
| 持仓股出现黑天鹅 | 立即评估影响 |

### 黄色预警（当日关注）

| 触发条件 | 应对 |
|---------|------|
| 持仓股涨跌 > 3% | 关注走势持续性 |
| 持仓股成交量异常放大 | 检查是否有消息 |
| 大盘涨跌 > 1.5% | 评估市场情绪 |
| 行业出现重大政策 | 评估影响 |

### 绿色信号（机会关注）

| 触发条件 | 应对 |
|---------|------|
| 关注股突破关键位置 | 评估是否买入 |
| 持仓股缩量整理 | 可能是蓄势 |
| 行业出现利好 | 评估持仓加仓可能 |
| 大盘回暖 | 考虑增加仓位 |
