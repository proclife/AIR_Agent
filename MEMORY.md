## 🎯 投资方法论总结

### 核心原则

1. **走势为王** — 走势和想法矛盾时，尊重走势
2. **止损第一** — 高位破线必须走
3. **只买便宜的好东西** — 低位布局，耐心等待
4. **仓位管理** — 核心持仓不超过50%，单只不超过20%

### 行业优先级

1. 新能源车（比亚迪）
2. AI人工智能（光模块、液冷、医疗）
3. 小金属（锂、锑）
4. 商业航天
5. 储能
6. 半导体

### 不碰的

- 纯概念炒作（无业绩）
- 高位机构抱团
- 流动性差的仙股
- 可能有破产风险的问题股

## 🧠 投资认知进化

### 对"趋势"的理解
- 五日线之上可以格局
- 五日线之下不要幻想
- 突破是关键信号

### 对"抄底"的理解
- 跌到历史低位 + 缩量 = 可能是底部
- 基本面不能有致命问题
- 要有输时间不输钱的心态

### 对"逃顶"的理解
- 高位放量滞涨 = 危险信号
- 跌破五周线 = 必须走
- "山顶逻辑最好听的时候最危险"

### 对"仓位"的理解
- 高位股轻仓试探
- 低位股可以重仓
- 保留子弹应对极端机会
