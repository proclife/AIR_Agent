## 二、核心思维链模式

### 1. 买入逻辑链
```
走势突破 → 发现机会 → 验证逻辑 → 果断买入
```

**典型案例（华钰矿业）**：
1. 走势漂亮：缩量下跌后突破
2. 基本面验证：ST困境反转 + 金矿资产
3. 可转债保护：10几块钱的转债有100元兜底
4. 买入方式：正股 + 可转债两部分配置

### 2. 卖出逻辑链
```
高位破线 → 执行止损 → 不要格局 → 认错离场
```

**核心理念**：
- 高位股跌破趋势必须走
- "山顶做梦是散户亏钱本质"
- 走势和想法矛盾时，尊重走势

### 3. 行业选择逻辑
```
政策支持 → 资金流入 → 走势验证 → 龙头介入
```

**优先级**：
1. 景气度高的赛道（新能源、AI）
2. 困境反转的低位股
3. 有实质性突破的技术/产品

## 三、关键判断标准

### 抄底标准
- 跌到历史低位
- 成交量极度萎缩
- 基本面没有破产风险
- "输时间不输钱"

### 逃顶标准
- 涨了10倍8倍的高位股
- 出现放量滞涨
- 跌破5日线/10日线
- "山顶逻辑最好听的时候最危险"

### 格局标准
- 五日线之上可以格局
- 低位股可以熬
- 高位股破线必须走
- "尊重走势，放弃幻想"

## 四、经典案例分析

### 案例1：华钰矿业（锑矿）
**买入时点**：ST后连续跌停，有人绝望割肉时
**买入逻辑**：
1. 走势突破买点
2. 困境反转（金矿 + 锑矿）
3. 可转债兜底保护
4. 小市值，弹性大

### 案例2：融捷股份（锂矿）
**操作**：2019年买入，持有2年
**逻辑**：
- 锂电景气度持续
- 走势一直在五日线之上
- 最终收益：约10倍

### 案例3：比亚迪
**核心理念**：
- "比亚迪永远的神"
- 每次大跌都是买入机会
- 跟踪趋势，不猜顶

## 五、风险控制思维

### 仓位管理
- 核心持仓：不超过50%
- 单只股票：不超过20%
- 保留现金：应对极端情况

### 止损原则
- 高位股：破线即走
- 低位股：可以等待
- 亏损底线：-10%必须反思

### 心态管理
- "输时间不输钱"
- "尊重走势，不要幻想"
- "高位格局最危险"
