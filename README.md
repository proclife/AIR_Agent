# AIR_Agent - 莫大韭菜投资研究项目

基于 OpenClaw 的投资推理助手，学习"莫大韭菜"的投资思维框架。

## 🎯 项目概述

这是一个自主运行的 AI 投资助手，能够：
- 自动监控持仓股走势
- 分析市场机会
- 生成投资建议
- 持续学习和进化

## 📂 项目结构

```
├── SOUL.md                    # AI推理灵魂
├── MEMORY.md                  # 投资记忆库
├── HEARTBEAT.md               # 市场监控系统
├── README.md                  # 本文件
│
├── 00_项目进度/               # 项目总结、进度追踪
├── 01_原始资料/               # 原始数据
│   ├── B站音频/               # B站直播音频（mp3）
│   ├── B站动态/               # B站动态数据
│   ├── B站转写/               # Whisper语音转写（原始）
│   ├── B站转写_可读版/        # 暗号替换后的可读版
│   ├── 数据管道/              # 数据抓取、转换、索引更新脚本
│   └── 索引文件/              # 代号→股票对照表
├── 02_市场数据/               # 历史行情、财报、重大新闻
├── 03_分析报告/               # 持仓分析、直播解读、定期报告
├── 04_验证校准/               # 回测、验证报告
│
└── backup/                    # 归档备份
```

**目录序号说明：**
- 0x = 项目管理与元信息
- 1x = 原始素材（音频、视频、直播转写、数据管道、索引）
- 2x = 外部市场数据
- 3x = 分析与研究成果
- 4x = 验证与校准

## 🧠 核心框架

### 三大铁律
1. **走势为王** - 走势和想法矛盾时，永远尊重走势
2. **止损第一** - 高位破线必须走
3. **只买便宜的好东西** - 低位布局，耐心等待

### 行业偏好
- 新能源车（比亚迪）
- AI人工智能（光模块、液冷、医疗）
- 小金属（锂、锑）
- 商业航天
- 储能

### 当前持仓（2026年4月）
锂矿(西藏矿业) > 氦气(华特气体) > 储能+比亚迪 > 商业航天 > AI液冷 > 锑(华钰矿业) > 磷 > AI医疗
