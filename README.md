# AIR_Agent - 自主投资推理Agent

基于 OpenClaw 的投资推理助手，学习"莫大韭菜"的投资思维框架。

## 🎯 项目概述

这是一个自主运行的 AI 投资助手，能够：
- 自动监控持仓股走势
- 分析市场机会
- 生成投资建议
- 持续学习和进化

## 📂 项目结构

```
├── SOUL.md              # AI推理灵魂
├── MEMORY.md            # 投资记忆库  
├── HEARTBEAT.md         # 市场监控系统
├── reports/             # 分析报告
├── data-pipeline/       # 数据管道
├── validation/          # 验证校准
└── progress/            # 项目进度
```

## 🧠 核心框架

### 三大铁律
1. **走势为王** - 走势和想法矛盾时，永远尊重走势
2. **止损第一** - 高位破线必须走
3. **只买便宜的好东西** - 低位布局，耐心等待

### 行业偏好
- 新能源车（比亚迪）
- AI人工智能（光模块、液冷、医疗）
- 小金属（锂、锑）
- 商业航天
- 储能

## ⚠️ 风险提示

本项目仅供学习和研究使用，不构成任何投资建议。投资有风险，决策需谨慎。
